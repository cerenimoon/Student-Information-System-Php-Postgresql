# [Student-Information-System-Php-PostgreSQL]

## 📌 Overview - Genel Bakış
- This repository features Student-Information-System project which inclides creating courses, enrolling or dropping courses as a student and checking information.
- Bu kod tabanı, Öğrenci-Bilgi-Sistemi projesini içermektedir, bağlantılı öğrenci bilgi sisteminde kurslar yaratılabilmektedir, kurslara kayıt yapılabilmektedir ya da kurs kayıtları iptal edilebilmektedir ve kurs bilgileri incelenebilmektedir. 

## 🚀 Key Features - Anahtar Özellikler
- **Code Area - Kod Alanı:** PHP, SQL
- **Robust Backend - Güçlü Arkaplan:** Integrated with a optimized PostgreSQL style relational database. - PostgreSQL tabanlı ilişkisel veri tabanıyla bağlantılıdır.  

## 🛠️ Tech Stack & Architecture - Teknik Özellikler ve Mimari
- **Batabase - Veritabanı:** PostgreSQL
- **DevOps / Infrastructure:**  Git, Github
